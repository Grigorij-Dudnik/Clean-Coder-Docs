AI that works whie you sleep? Not yet, but we endeavor to it. Althrough, the first step has already been taken - we have AI that works while you work.

## How to use

When working with Manager agent and working on the task, Clean Coder automatically starting working on the next task in the background. It doesn't complete entire pipeline, but file research only. When you start work on the next task, your file research will be already done, saving your time.
